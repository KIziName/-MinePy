**MinePy – 2D песочница на Python**

MinePy — это 2D-игра с бесконечным процедурным миром, сменой дня и ночи, системой крафта и враждебными мобами. Полностью на Python + Pygame.

**Особенности**

· 🗺️ Генерация чанков с холмами, рудами (медь, железо, золото, уголь) и деревьями.

· 🌤️ Динамическая погода (дождь/снег) и цикл дня/ночи (смена неба, появление разных мобов).

· ⚔️ Боевая система: днём — слаймы, ночью — зомби, летающие глаза и скелеты.

· 🎒 Инвентарь с перетаскиванием предметов, крафт оружия, зелий, брони и факелов.

· 💾 Автосохранение прогресса (карта, инвентарь, мобы, позиция) в %AppData%/MinePy.

**Управление**

WASD / Стрелки — движение, Space — прыжок, E — инвентарь, Esc — пауза,
ЛКМ — атака/разрушение, ПКМ — установка блока, 1–0 — выбор слота.

⚠️ Проект в стадии альфы, возможны баги.
